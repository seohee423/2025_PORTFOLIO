  • 24/05/07 - 24/05/23

LILYBYRED는 다양한 매력을 컬러로 완성하는 뷰티 브랜드입니다. 다양한 디바이스에서 이용할 수 있도록 온라인 쇼핑몰을 반응형 웹으로 구현하고, 다채로운 이벤트를 통해 릴리바이레드만의 사랑스럽고 생기 넘치는 분위기를 전달합니다.

Goals

1. 학습한 내용을 적용해 웹 사이트를 구현하며 퍼블리싱에 대한 이해도, 숙련도 향상

2. GitHub, Git Organizaion을 활용한 협업능력 강화

3. javaScript, jQuery를 이용한 고난도 반응형 웹사이트 구현

Result

GOOD
  • 기한 내에 디자인, 퍼블리싱 완료
  • 맡은 부분이 아니더라도 적극적으로 도움
  • GitHub Organization으로 효율적인 협업
  • 구성원 간의 충분한 소통으로 통일된 코드

BAD
  • 서브페이지, 메인 페이지 디자인이 유사함
  • 기간 내 마무리하기 위해 다양한 기종별로 달라지는 여백, 간격 등 세밀한 부분을 수정하지 못한 것
  • 이벤트 구현에 미숙해 적극 참여하지 못한 점
REVIEW

릴리바이레드 반응형 웹 사이트를 디자인, 퍼블리싱하는 팀 프로젝트에 참여했습니다.
프로젝트를 통해 퍼블리싱에서 협업할 때에는 Organization을 활용해 작업 효율성을 높인다는 점을 배웠습니다. JSON, AJAX 등 새로운 기술을 배우고 적용하는 과정이 쉽지 않았지만 이를 계기로 팀원들과 함께 공부하며 퍼블리싱 작업에서 부족했던 부분을 인식/보완해 한층 발전할 수 있었습니다.
늘 칭찬이 오가던 화기애애한 분위기와 구성원들의 열정 넘치는 모습이 기억에 남습니다.
